# Zetech Lost & Found Student IDs Online Platform

A web-based system designed to help students report lost student IDs and track found IDs within Zetech University.  
This platform provides students with an easy way to report lost IDs and helps staff manage and update found items efficiently.

---

## Features
- **Student Registration & Login:** Secure login for students to access their dashboard  
- **Report Lost ID:** Students can submit lost ID reports with details  
- **View Found IDs:** Students can check IDs that have been found and reported  
- **Admin Dashboard:** Admins can approve lost/found reports, update status, and manage records  
- **Status Tracking:** Students can see if their report is pending or approved  
- **Notifications (Optional):** Notify students when their lost ID is found

---

## Technologies Used
- **Backend:** PHP  
- **Database:** MySQL  
- **Frontend:** HTML, CSS, JavaScript  
- **Tools:** XAMPP, phpMyAdmin

---

## How to Run the Project
1. Install [XAMPP](https://www.apachefriends.org/) or any local PHP/MySQL server.  
2. Copy the project folder to your `htdocs` directory.  
3. Start **Apache** and **MySQL** via XAMPP control panel.  
4. Import the database file located at `/sql/lost_found_db.sql` using phpMyAdmin.
